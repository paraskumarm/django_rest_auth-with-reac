# django_rest_auth-with-react
<br />
To run backend - 
<br />
cd backend
<br />
pipenv shell
<br />
pip3 install -r requirements.txt
<br />
cd App
<br />
python3 manage.py makemigrations
<br />
python3 manage.py migrate
<br />
python3 manage.py runserver
<br />


To run frontend - 
<br />
cd frontend/app
<br />
npm start
